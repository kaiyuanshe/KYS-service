# KaiYuanShe service

**RESTful API** service of [KaiYuanShe][1]

## Development

### Installation

```shell
npm i pnpm -g
pnpm i
```

### Start Dev Env

```shell
pnpm dev
```

### Build Docker Image

```shell
pnpm dock
```

### Run Docker Image

```shell
pnpm docker
```

[1]: https://kaiyuanshe.cn
