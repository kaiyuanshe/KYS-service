# KaiYuanShe service

**RESTful API** service of [KaiYuanShe][1]

[![Deploy to Production environment](https://github.com/kaiyuanshe/KYS-service/actions/workflows/deploy-production.yml/badge.svg)][2]

## Development

### Installation

```shell
npm i pnpm -g
pnpm i
```

### Start Dev Env

```shell
pnpm dev
```

### Build Docker Image

```shell
pnpm dock
```

### Run Docker Image

```shell
pnpm docker
```
